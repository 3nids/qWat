QWAT Installation Guide
=======================


This will guide you through the setup of required applications and services to
get your system ready for running QWAT.

.. note::

 You also have the option of downloading a `Demo-VM` that has a (not updated) working QWAT/QGEP demo. See `QGEP documentation <http://qgep.github.io/docs/vm/index.html>`_.
